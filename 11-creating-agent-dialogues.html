Creating Agent Dialogues

Activity Guidance

Create an agent dialogue, using KQML and KIF, between two agents (named Alice and Bob).

Alice is an agent designed to procure stock and Bob is an agent that controls the stock levels for a warehouse. This dialogue should see Alice asking Bob about the available stock of 50 inch televisions, and also querying the number of HDMI slots the televisions have.

    Alice: Stock Procurement Agent
    Bob: Warehouse Stock Control Agent
    Dialogue:
    1. Alice to Bob:
    KQML: 
          (ask-if 
              :sender Alice 
              :receiver Bob
              :reply-with  Q001
              :language KQML 
              :ontology warehouse-inventory
              :content ((item television) (size 50) )
          )
    KIF: ((isa ?item television) (= (size ?item) 50))
    Meaning: Alice asks Bob if there are any televisions (item) of size 50 inches available in the warehouse inventory.

    2. Bob response to Alice
    KQML: 
            (reply 
                :sender Bob 
                :receiver Alice
                :in-reply-to  Q001
                :language KQML :ontology warehouse-inventory
                :content (and (item television) (size 50) (quantity 10))
            )
    Meaning: Bob confirms there are televisions (item) of size 50 inches available (quantity 10) in the warehouse

    3. Alice inquires about features
    KQML: 
            (ask-if 
                :sender Alice :receiver Bob
                :in-reply-to Q001
                :language KQML :ontology television-features
                :content (and (item television) (size 50) (has-feature hdmi))
            )
    KIF: (and (isa ?item television) (= (size ?item) 50) (has-feature ?item hdmi))
    Meaning: Alice asks Bob if the 50-inch televisions (item) have an HDMI feature (has-feature hdmi).

    4. Bob responds with feature information
    KQML: 
            (reply 
                :sender Bob 
                :receiver Alice
                :in-reply-to Q001
                :language KQML :ontology television-features
                :content (and (item television) (size 50) (has-feature hdmi) (number-of-hdmi 2))
            )
    KIF: (and (isa ?item television) (= (size ?item) 50) (has-feature ?item hdmi) (= (number-of-hdmi ?item) 2))
    Meaning: Bob confirms the 50-inch televisions (item) have an HDMI feature (has-feature hdmi) and have 2 HDMI slots (number-of-hdmi 2)

This dialogue demonstrates how KQML and KIF can be used for communication between agents. Alice uses KQML performatives to ask questions and Bob replies with relevant information using KIF expressions within the specified ontology.
